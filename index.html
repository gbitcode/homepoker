How to Use

Setup

  • Set starting balance and blinds
  • Click "Start New Game"
  • Players join by clicking empty seats

Game Flow

  • 2+ players needed to start a hand
  • Click "Start Hand" when ready
  • Blinds are posted automatically
  • Active player has buttons: Fold, Call/Check, Raise, All In

Phases

  • Waiting - Join/leave, adjust blinds, refill balance
  • Pre-flop - After blinds, first betting round
  • Flop - Second betting round
  • Turn - Third betting round
  • River - Final betting round

Controls

  • +/- Blinds - Adjust blind levels (only before hand)
  • End Hand - Manually select winner
  • Refill - Add balance to player (only before hand)
  • End Game - Return to setup screen
